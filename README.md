# Predictive-Modeling-ML

## Project Overview

This project uses Machine Learning to predict passenger survival on the Titanic dataset.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-Learn
* Jupyter Notebook

## Algorithm Used

Decision Tree Classifier

## Steps Performed

1. Data Loading
2. Data Cleaning
3. Feature Selection
4. Label Encoding
5. Train-Test Split
6. Model Training
7. Accuracy Evaluation
8. Confusion Matrix Visualization
9. ROC Curve Analysis

## Results

* Accuracy: 75.42%
* Model: Decision Tree Classifier

## Outcome

Successfully implemented a supervised machine learning model and evaluated its performance using accuracy, confusion matrix, and ROC curve.
